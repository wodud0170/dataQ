원활한 BackEnd 작업을 위해 vue 설치는 아래와 같이 진행합니다.
(참고: lint, test 등등은 사용하지 않습니다.)

1. vue-CLI 전역 설치
2. 프로젝트 생성


```
npm install -g vue-cli

vue init webpack my-project
```


### Project setup
