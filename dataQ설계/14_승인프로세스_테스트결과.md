## 3. 추가 확인 필요 사항

### 3.1 미승인 단어 체크 로직 테스트 보완
- 유효한 도메인을 포함한 테스트 케이스로 재테스트 필요
- 현재 테스트는 도메인 FK 에러가 먼저 발생하여 승인 체크 로직까지 도달 못함

### 3.2 실패 건 이력 저장 정책
- 현재: 실패한 일괄등록도 이력에 남음 (성공:0, 실패:1)
- 검토: 실패 건은 이력에 남기지 않는 것이 맞을 수 있음

### 3.3 DB 레벨 승인 상태 확인
```sql
-- 테스트로 등록한 단어의 승인 상태 확인
SELECT WORD_NM, WORD_ENG_ABRV_NM, APRV_YN 
FROM TB_WORD 
WHERE WORD_ENG_ABRV_NM = 'TSTVRFY';
```
