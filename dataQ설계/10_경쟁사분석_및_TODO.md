## 1. 경쟁사 솔루션 개요

### 비투엔 SMETA (에스메타)

| 항목 | 내용 |
|------|------|
| 개발사 | (주)비투엔컨설팅 |
| 포지셔닝 | 메타데이터 표준 관리 솔루션 (공공부문 1위) |
| 주요 고객 | 환경부, 한국전력공사, 우체국 차세대 등 |
| 시장점유율 | 조달청 디지털서비스몰 메타관리 67% (2023) |
| 인증 | GS인증 1등급 |
| 기술 스택 | Java, 웹 기반 |
| 관련 솔루션 | SDQ(품질관리), SFLOW(데이터통합), BizMeta(비즈니스 메타) |

**주요 기능 (사용자 매뉴얼 V2.1 기준 상세):**

**표준사전 관리:**
- 표준등록: 도메인/단어/용어를 등록 요청 (결재 프로세스 필수)
- 도메인: 신규/변경/삭제 요청, 코드 도메인(공통코드/개별코드/계층코드/통합코드), 유효값 관리, 분류어 제한
- 단어: 신규/변경/삭제 요청, 단어유형(기본어/분류어), 행정표준용어 참조
- 용어: 신규/변경/삭제 요청, 용어명 입력 시 표준단어 자동분할, 비표준사유 입력, 금칙어/유사어 자동 경고
- 코드 조회/관리
- 도메인그룹 관리, 데이터타입 관리 (논리/물리 타입 매핑)
- 금칙/유사어 관리 (엑셀 등록)
- 행정용어 관리 (엑셀 등록)
- 이력정보: 변경 시마다 이력 저장, 이력 간 비교 기능
- 영향도 분석: 도메인/단어/용어 변경 시 연관 용어/엔터티/테이블 목록 조회

**데이터베이스 관리:**
- 테이블/인덱스 등록 (결재 프로세스)
- 테이블/인덱스 조회

**데이터모델 관리:**
- 주제영역 관리
- 엔터티 등록/조회 (결재 프로세스)
- 논리모델 기반 물리모델 자동 생성

**결재 프로세스:**
- 모든 등록/변경/삭제가 결재를 통해 진행 (내요청목록, 내결재목록)
- 다단계 결재, 반려, 결재취소, 재요청
- 결재 정보 및 이력 조회

**표준화지원도구:**
- 스키마 수집: DBMS에 접속하여 테이블/컬럼 정보 수집 (테이블 단위 선택 가능)
- 표준단어 일괄등록: 수집된 컬럼을 _ 기준 분리 → 미등록 단어에 한글명 작성 → 일괄 등록
- 엑셀 일괄등록: 엑셀 용어 업로드 → 형태소 자동분할(행정표준용어/KOMORAN) → 단어 일괄 등록
- 표준용어 일괄등록: 수집 컬럼 기반 용어 자동 생성, 도메인 일괄 지정, 용어 일괄 등록
- 테이블 일괄등록: 수집 테이블 일괄 등록

**통계:**
- 표준분석결과: 시스템별 표준 준수율/적용률 차트 + 불일치 컬럼 목록
- 테이블분석결과: DBMS vs 메타시스템 테이블/인덱스 일치율 + 컬럼 상세 비교 팝업
- 엔터티분석결과: DBMS vs 메타시스템 엔터티 일치율 + 속성 상세 비교

**대시보드:**
- 표준 준수/적용률 차트
- DBMS vs 메타시스템 일치율 원형 차트
- 월별 요청 건수, 표준메타 건수
- 요청/결재 목록, 바로가기, 공지사항

**기타:**
- 통합검색 (도메인/단어/용어/엔터티/테이블/인덱스)
- 게시판 (공지사항, Q&A)
- 엑셀 업로드/다운로드 (모든 메뉴에서 지원)
- 멀티시스템 (로그인 시 시스템 선택, 화면에서 변경 가능)
- Eclipse 플러그인 (용어 검색)
- 지원 DBMS: Oracle, Tibero (매뉴얼 기준)

| 인증 | GS인증 1등급, 가트너 뉴스레터 등재 |

**주요 기능:**

- 코드 통합 관리
- DB 카탈로그 수집 (Oracle, SQL Server, DB2, Tibero, MariaDB 등)
- 전자 결재 시스템 연동
- 통합 메타데이터 저장소
- 데이터 거버넌스 포털 (검색, 변경 요청, 협업)
- 역할별 업무흐름 (개발자, 모델러, DBA, CIO)
- 범정부 메타데이터 연계
- 개인화 관리 기능

| 항목 | 내용 |

**주요 기능:**
- 데이터 표준화 관리 (단어/용어/도메인/유사어/공통코드 통합 관리)

- 표준준수 현황 모니터링

- DB 메타데이터 추출 (테이블/컬럼/인덱스 오브젝트 정보 + 변경 이력 추적)
- 멀티표준 지원 (업무별/시스템별 여러 표준 관리)
- 표준화 메타데이터 배포 자동화
- 빅데이터 플랫폼 지원 (Hive, Impala)
- 클라우드 연동 (NCloud, AWS S3, Redshift)
- 영향도 분석 (ImpactStream — 별도 제품으로 제공)
- 모델 뷰어 및 관계 정보 시각화
- 데이터 거버넌스 포털 (IRUDA — 별도 제품)

## 3. Narae DataQ의 차별점 (강점)

### 3.1 자동 표준화 추천 시스템
경쟁사에 없는 **독자적인 기능**:
- OKT 형태소 분석 + greedy longest-match로 입력 용어를 자동 분리
- TB_WORD(표준단어) > TB_WORD_DICT(추천사전) > OKT 토큰 우선순위
- **다중 분리 추천** (1순위: 의미단위 최대, 2순위: 최소단위 추가분리)
- 단어 선등록 → 용어 등록 프로세스
- SMETA/WISE META 모두 이 수준의 자동화는 제공하지 않음
- WISE META의 Data Butler가 GPT 기반 추천을 하지만, 우리는 **오프라인/온프레미스** 환경에서 외부 API 없이 동작

### 3.2 진단 결과 → 즉시 등록 연계
- 진단 결과에서 바로 일괄 용어등록 가능
- 일괄 코멘트 스크립트 생성 (DBA 작업용)
- 경쟁사는 진단과 등록이 별도 메뉴

### 3.3 경량 아키텍처
- Spring Boot + Vue.js + PostgreSQL로 빠른 도입 가능
- 별도 미들웨어/WAS 불필요

## 6. 전략적 포지셔닝 제안

### 현재 포지션
- **강점**: 자동 표준화 추천 (경쟁사 대비 독보적), 진단→등록 즉시 연계, 경량 아키텍처
- **강점 추가**: 구조 진단 (수집 스냅샷 vs 실제 DBMS 라이브 비교) — 구현 완료
- **약점**: Lineage/영향도 분석 등 엔터프라이즈 기능 부재

### 권장 전략

**"자동 표준화" 를 핵심 차별점으로 내세우고, 기본 기능 GAP을 빠르게 메우는 전략**

1. **Phase 1** 완료 후 공공 입찰 참여 가능 수준 달성
2. 자동 표준화 추천의 우수성을 BMT/PoC에서 시연 (경쟁사 대비 명확한 차별점)
3. 온프레미스/오프라인 동작을 장점으로 포지셔닝 (보안 요구 고객 대응)
4. 장기적으로 LLM 추천은 옵션으로 제공 (WISE META Data Butler 대응)
