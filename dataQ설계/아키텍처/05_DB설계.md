# 05. DB 설계

> 최종 수정: 2026-03-23
> 내부 DB: **PostgreSQL**

## DDL 실행 순서

```
1. TB_SYS_INFO
2. TB_DATA_SOURCE
3. TB_DOMAIN_GROUP (논리, 실제로는 TB_DOMAIN에서 집계)
4. TB_DOMAIN
5. TB_WORD
6. TB_TERMS
7. TB_CODE_INFO
8. TB_CODE_DATA
9. TB_APPROVE_STAT
10. TB_DATA_MODEL
11. TB_DATA_MODEL_CLCT
12. TB_DATA_MODEL_SCHEMA
13. TB_DATA_MODEL_OBJ
14. TB_DATA_MODEL_ATTR
15. TB_DATA_MODEL_STATS
16. TB_DIAG_JOB
17. TB_DIAG_RESULT
```

## 테이블 정의

### 시스템 / 데이터소스

#### TB_SYS_INFO (시스템 정보)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| SYS_CD | VARCHAR PK | 시스템 코드 |
| SYS_NM | VARCHAR | 시스템명 |
| SYS_DESC | VARCHAR | 설명 |
| CRET_DT | TIMESTAMP | 생성일시 |

#### TB_DATA_SOURCE (데이터소스)
> 스키마: `ndata`
| 컬럼 | 타입 | 설명 |
|------|------|------|
| DS_ID | VARCHAR PK | 데이터소스 ID |
| DSN | VARCHAR | 데이터소스명 |
| DBMS_TP | VARCHAR | DBMS 유형 (oracle/postgresql/mysql/mariadb/mssql/cubrid) |
| HOST | VARCHAR | 호스트 |
| PORT | VARCHAR | 포트 |
| DB_NM | VARCHAR | DB명 |
| USER_NM | VARCHAR | 접속 사용자명 |
| PWD | VARCHAR | 비밀번호 (Jasypt 암호화) |
| SYS_CD | VARCHAR FK | 시스템 코드 |
| CRET_DT | TIMESTAMP | 생성일시 |

### 표준 사전

#### TB_DOMAIN (도메인)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| DOMAIN_ID | BIGSERIAL PK | 도메인 ID |
| DOMAIN_NM | VARCHAR | 도메인명 |
| DOMAIN_GRP_NM | VARCHAR | 도메인 그룹명 |
| DOMAIN_CLSF_NM | VARCHAR | 도메인 분류명 |
| DOMAIN_DESC | TEXT | 설명 |
| DATA_TYPE | VARCHAR | 데이터 타입 |
| DATA_LEN | INTEGER | 길이 |
| DATA_DECIMAL_LEN | INTEGER | 소수 자릿수 |
| DATA_UNIT | VARCHAR | 단위 |
| STOR_FMT | VARCHAR | 저장 형식 |
| EXPR_FMT_LST | TEXT | 표현 형식 목록 |
| ALLOW_VAL_LST | TEXT | 허용값 목록 |
| COMM_STND_YN | CHAR(1) | 공통표준 여부 (Y/N) |
| MAGNTD_ORD | INTEGER | 크기 순서 |
| APRV_YN | CHAR(1) | 승인 여부 (Y/N) |
| APRV_USER_ID | VARCHAR | 승인자 ID |
| REQ_SYS_CD | VARCHAR | 요청 시스템 코드 |
| CRET_DT | TIMESTAMP | 생성일시 |
| CRET_USER_ID | VARCHAR | 생성자 ID |
| UPDT_DT | TIMESTAMP | 수정일시 |
| UPDT_USER_ID | VARCHAR | 수정자 ID |

#### TB_TERMS (용어)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| TERMS_ID | BIGSERIAL PK | 용어 ID |
| TERMS_NM | VARCHAR | 용어명 (한글) |
| TERMS_ENG_ABRV_NM | VARCHAR | 영문 약어명 |
| TERMS_DESC | TEXT | 설명 |
| DOMAIN_NM | VARCHAR | 도메인명 (TB_DOMAIN 참조) |
| ALLOPH_SYNM_LST | TEXT | 동의어/이음이의어 목록 |
| TOKEN_LST | TEXT | 형태소 분석 토큰 목록 |
| APRV_YN | CHAR(1) | 승인 여부 |
| CRET_DT | TIMESTAMP | 생성일시 |
| CRET_USER_ID | VARCHAR | 생성자 ID |

#### TB_WORD (단어)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| WORD_ID | BIGSERIAL PK | 단어 ID |
| WORD_NM | VARCHAR | 단어명 |
| WORD_ENG_NM | VARCHAR | 영문명 |
| WORD_MEANING | TEXT | 의미 |
| WORD_CATEGORY | VARCHAR | 범주 |
| APRV_YN | CHAR(1) | 승인 여부 |
| CRET_DT | TIMESTAMP | 생성일시 |
| CRET_USER_ID | VARCHAR | 생성자 ID |

#### TB_CODE_INFO (코드 정보)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| CODE_ID | VARCHAR PK | 코드 ID |
| CODE_NM | VARCHAR | 코드명 |
| CODE_DESC | TEXT | 설명 |
| CRET_DT | TIMESTAMP | 생성일시 |

#### TB_CODE_DATA (코드값)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| CODE_DATA_ID | BIGSERIAL PK | 코드값 ID |
| CODE_ID | VARCHAR FK | 코드 ID |
| CODE_VALUE | VARCHAR | 코드값 |
| CODE_DESC | TEXT | 설명 |
| SORT_ORD | INTEGER | 정렬 순서 |

#### TB_APPROVE_STAT (승인 상태)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| APPROVE_ID | BIGSERIAL PK | 승인 ID |
| OBJ_TYPE | VARCHAR | 객체 타입 (DOMAIN/TERMS/WORD 등) |
| OBJ_ID | VARCHAR | 객체 ID |
| APRV_YN | CHAR(1) | 승인 여부 |
| APRV_USER_ID | VARCHAR | 승인자 ID |
| APRV_DT | TIMESTAMP | 승인일시 |

### 데이터모델

#### TB_DATA_MODEL (데이터모델)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| DM_ID | VARCHAR PK | 데이터모델 ID |
| DM_NM | VARCHAR | 데이터모델명 |

| CRET_DT | TIMESTAMP | 생성일시 |

| 컬럼 | 타입 | 설명 |

| CLCT_START_DT | TIMESTAMP | 수집 시작일시 |

| 컬럼 | 타입 | 설명 |
|------|------|------|
| DM_ID | VARCHAR FK | 데이터모델 ID |
| SCHEMA_NM | VARCHAR | 수집 대상 스키마명 |

|------|------|------|
| OBJ_ID | BIGSERIAL PK | 객체 ID |
| DM_CLCT_ID | VARCHAR FK | 수집 ID |
| OBJ_NM | VARCHAR | 테이블명 |
| OBJ_NM_KR | VARCHAR | 한글 테이블명 |

| SCHEMA_NM | VARCHAR | 스키마명 |

#### TB_DATA_MODEL_ATTR (컬럼 정보)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| ATTR_ID | BIGSERIAL PK | 속성 ID |
| DM_CLCT_ID | VARCHAR FK | 수집 ID |
| OBJ_NM | VARCHAR | 테이블명 |
| ATTR_NM | VARCHAR | 컬럼명 (영문) |
| ATTR_NM_KR | VARCHAR | 컬럼 한글명 |
| DATA_TYPE | VARCHAR | 데이터 타입 |

| DATA_DECIMAL_LEN | INTEGER | 소수 자릿수 |

| PK_YN | CHAR(1) | PK 여부 |
| FK_YN | CHAR(1) | FK 여부 |
| DEFAULT_VAL | VARCHAR | 기본값 |
| TERMS_STND_YN | CHAR(1) | 용어 표준화 여부 |
| WORD_LST | TEXT | 단어 목록 |
| WORD_STND_LST | TEXT | 표준화 단어 목록 |
| SORT_ORD | INTEGER | 정렬 순서 |

|------|------|------|

|------|------|------|
| DIAG_JOB_ID | VARCHAR PK | 진단 JOB ID (UUID) |

| DATA_MODEL_NM | VARCHAR | 데이터모델명 |
| CLCT_DT | VARCHAR | 수집일시 |
| STATUS | VARCHAR | 상태 (READY/RUNNING/DONE/STOPPED/ERROR) |
| TOTAL_CNT | INTEGER | 전체 컬럼 수 |
| PROCESS_CNT | INTEGER | 처리된 컬럼 수 |
| RESULT_CNT | INTEGER | 이슈 발견 수 |
| CRET_DT | TIMESTAMP | 생성일시 |
| CRET_USER_ID | VARCHAR | 생성자 ID |
| START_DT | TIMESTAMP | 진단 시작일시 |
| END_DT | TIMESTAMP | 진단 종료일시 |

| 컬럼 | 타입 | 설명 |
|------|------|------|

| DIAG_JOB_ID | VARCHAR FK | 진단 JOB ID |
| OBJ_NM | VARCHAR | 테이블명 |
| ATTR_NM | VARCHAR | 컬럼명 (영문) |
| ATTR_NM_KR | VARCHAR | 컬럼 한글명 |
| DIAG_TYPE | VARCHAR | 진단 유형 |
| DIAG_DETAIL | TEXT | 상세 설명 |
| STD_VALUE | VARCHAR | 표준 기준값 |
| ACTUAL_VALUE | VARCHAR | 실제값 |

## 관계도 (간략)

```
TB_SYS_INFO
    └─ TB_DATA_SOURCE (SYS_CD FK)
            └─ TB_DATA_MODEL (DM_DS_ID FK)
                    ├─ TB_DATA_MODEL_SCHEMA (DM_ID FK)
                    └─ TB_DATA_MODEL_CLCT (DM_ID FK)
                            ├─ TB_DATA_MODEL_OBJ (DM_CLCT_ID FK)
                            ├─ TB_DATA_MODEL_ATTR (DM_CLCT_ID FK)
                            ├─ TB_DATA_MODEL_STATS (DM_CLCT_ID FK)
                            └─ TB_DIAG_JOB (CLCT_ID FK)
                                    └─ TB_DIAG_RESULT (DIAG_JOB_ID FK)

TB_DOMAIN ◀─ TB_TERMS (DOMAIN_NM 참조)
TB_WORD
TB_CODE_INFO ─ TB_CODE_DATA (CODE_ID FK)
TB_APPROVE_STAT (OBJ_TYPE + OBJ_ID로 다형성 참조)
```
