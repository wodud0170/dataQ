## 자동화 방법

모든 시나리오는 Selenium Edge WebDriver로 자동화 가능:

```python
# 공통 패턴
driver.get(BASE + '/signin')         # 페이지 이동
driver.find_element(By.CSS_SELECTOR, ...).send_keys(...)  # 입력
driver.find_element(By.CSS_SELECTOR, ...).click()         # 클릭
driver.execute_script("...")          # Vue 컴포넌트 직접 호출
assert '기대텍스트' in driver.page_source  # 검증
driver.save_screenshot(...)          # 증빙 스크린샷
```
