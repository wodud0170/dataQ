# 소스분석 - 죽은 기능 (미구현 / 주석 처리)

> 최종 수정: 2026-03-23

## 개요

소스에 존재하지만 현재 주석 처리되었거나, 화면은 있으나 백엔드 연결이 끊긴 기능 목록.
향후 부활시킬 가능성이 있는 것과 완전히 제거해도 무방한 것으로 구분.

---

## 1. 데이터 품질 관리 (DQ 기능 전체)

### 상태
NdNav.vue, NdContent.vue에서 **완전 주석 처리**.
Vue 컴포넌트 파일은 존재하나 메뉴에서 접근 불가.

### 해당 컴포넌트
| 컴포넌트 | 기능 | 탭 ID |
|---------|------|-------|
| `DQDQI.vue` | 데이터 품질지표(DQI) 관리 | dqi |
| `DQCTQ.vue` | 핵심관리항목(CTQ) 관리 | ctq |
| `DQBR.vue` | 업무규칙(Business Rule) 관리 | dqbr |
| `DQQVTarget.vue` | 품질 검증 대상 설정 | target |
| `DQQVQV.vue` | 품질 검증 실행 | qv |
| `DQQVRVI.vue` | 검증항목별 결과 조회 | rvi |
| `DQQVRT.vue` | 테이블별 결과 조회 | dqqvrt |

### 의도
데이터 품질 관리는 dataQ의 핵심 목표 기능으로 보임.
표준 사전 → 데이터모델 수집 → **표준화 진단** → **품질 검증** 순으로 이어지는 파이프라인.
현재는 표준화 진단까지만 구현, 품질 검증(DQ) 단계 미착수.

### 부활 가능성
**높음** - 시스템의 최종 목표 기능으로 추후 구현 예정으로 보임.

## 2. 진단 현황 (AND 그룹)

### 상태
NdNav.vue에서 주석 처리.

### 해당 컴포넌트
| 컴포넌트 | 기능 |
|---------|------|
| `ANDscurrent.vue` | 데이터 표준화 현황 (구버전 진단 화면으로 추정) |
| `ANDqcurrent.vue` | 데이터 품질 현황 |

### 의도
현재 구현된 `DSDataDiag.vue` / `DSDataDiagResult.vue`로 대체된 것으로 추정.
`ANDscurrent.vue`는 구버전 진단 화면의 잔재일 가능성이 높음.

### 부활 가능성
**낮음** - `DSDataDiag.vue`로 대체됨. 내용 검토 후 삭제 권장.

---

## 3. 역할 및 권한 관리 (MMRoles.vue)

### 상태
NdNav.vue 메뉴에서 `style="display:none"` 으로 **숨김 처리**.
컴포넌트 파일은 존재.

### 의도
사용자별 역할(Role) 부여 및 메뉴 접근 권한 관리.
현재는 모든 API가 `permitAll()`로 인증 없이 접근 가능하므로 권한 관리 의미 없는 상태.

### 부활 가능성
**중간** - 다중 사용자 환경에서는 필수. 현재는 내부 단일 팀 사용으로 보류된 것으로 추정.

---

## 4. 대시보드 (QDashboard.vue)

### 상태
컴포넌트 존재, 메뉴 접근 가능하나 **내용 구현이 미흡**.

### 현재 구현 내용
- 미승인 건수 표시
- 승인 탭 바로가기

### 미구현 내용
- 표준화율 추이 차트 (ApexCharts import는 되어 있음)
- 최근 수집 현황 요약
- 진단 결과 요약

### 부활/완성 가능성
**높음** - 진입 화면으로서 각 기능의 현황을 한눈에 보여주는 역할이 필요.

---

## 5. WebSocket 완료 메시지 기반 업로드 처리

### 상태
**기능 변경됨** (WebSocket → HTTP 동기 방식).

### 배경
일괄등록(Excel 업로드) 완료 신호를 WebSocket으로 수신하는 방식이었으나,
DB 처리 시간이 길어지면 STOMP 세션이 끊기거나 메시지가 유실되는 문제 발생.

### 현재 방식
`runService` (동기) → HTTP 응답으로 완료 신호 전달.

### 잔재 코드
main.js, NdNav.vue 하단의 WebSocket 로그 뷰어는 여전히 STOMP 메시지를 수신하나,
업로드 완료 메시지는 더 이상 오지 않음.

---

## 6. 통합 검색 (SearchController)

### 상태
백엔드 API (`/api/search/search`) 존재하나 **프론트엔드 연결 미확인**.

### 의도
용어·단어·도메인·코드·데이터모델을 한 번에 검색하는 통합 검색 기능.

### 부활 가능성
**중간** - UX 개선을 위해 헤더 검색바로 연결하면 유용.
